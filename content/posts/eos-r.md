ソニーのフルサイズミラーレス α シリーズがでて早数年が経ち、ようやく Nikon, Canon, Panasonic がフルサイズミラーレスに参入をしてきました。
今年発売されるのは、Nikon Z7, Z6, Canon EOS R です。

## EOS R を買う前の期待
<a data-flickr-embed="true"  href="https://www.flickr.com/photos/puzzel-fukuchi/31895059878/in/dateposted/" title="IMG_5730"><img src="https://farm5.staticflickr.com/4849/31895059878_7a29039cdf_b.jpg" width="1024" height="683" alt="IMG_5730"></a><script async src="//embedr.flickr.com/assets/client-code.js" charset="utf-8"></script>

EOS R の前より、Canon からミラーレスは発売されていました。80D のセンサーでありながら安価かつ軽量という点で EOS Kiss M は買おうか迷っていましたが、α7R3 や P20 Pro を購入したので、諦めました。なので、初のCanon のミラーレスということで、

- 軽量
- Canon の色
- Canon の操作
- 安定した EF レンズの使用

を主に期待していました。操作感については、α はまじで慣れなかった(タッチパネルがないことでさらに慣れなかった)事がありました。

## 買って使って思ったこと

<a data-flickr-embed="true"  href="https://www.flickr.com/photos/puzzel-fukuchi/45766578851/in/dateposted/" title="IMG_5732"><img src="https://farm5.staticflickr.com/4849/45766578851_a38720e003_b.jpg" width="1024" height="683" alt="IMG_5732"></a><script async src="//embedr.flickr.com/assets/client-code.js" charset="utf-8"></script>
<a data-flickr-embed="true"  href="https://www.flickr.com/photos/puzzel-fukuchi/45716334682/in/dateposted/" title="IMG_5722"><img src="https://farm2.staticflickr.com/1910/45716334682_27f5878f51_b.jpg" width="1024" height="683" alt="IMG_5722"></a><script async src="//embedr.flickr.com/assets/client-code.js" charset="utf-8"></script>


- 軽量っちゃ軽量
    - 5D と比べると軽いが、7R3 と比べると重い。
    - ただ、α7R3 の MC-11 と比べると コントロールリング マウントは軽いので結果的に軽くはなった…
- ちゃんと Canon の色が出る
    - α7R3 の時はやや鮮やか目な色だったんですが、慣れた Canon の色で撮れます。現像がしやすい、、、
- タッチパネルでメニュー操作はやっぱり楽。
    - 70D のときに体感はしていたんですが、ほぼ変わらずの操作感かつ、タッチパネルでいろいろ操作できるのは便利。
 - AF が早すぎ・安定して EF レンズが使える
    - 今のところレンズは、EF 24-105mm f4L と EF 50mm f1.2L しか使ってないですが、5D のときよりもピンが早い、動的なものを追っかけてくれることに感動してます。
    - α7R3 のときは、社外製というとことで、ピンずれや遅かったり満足はできていなかったので、AF 周りは満足しています。
   
## 微妙なところ
<a data-flickr-embed="true"  href="https://www.flickr.com/photos/puzzel-fukuchi/44853075195/in/dateposted/" title="IMG_5726"><img src="https://farm5.staticflickr.com/4829/44853075195_21869ea20c_b.jpg" width="1024" height="683" alt="IMG_5726"></a><script async src="//embedr.flickr.com/assets/client-code.js" charset="utf-8"></script>
<a data-flickr-embed="true"  href="https://www.flickr.com/photos/puzzel-fukuchi/31895059558/in/dateposted/" title="IMG_5733"><img src="https://farm2.staticflickr.com/1919/31895059558_eddf607a9c_b.jpg" width="1024" height="683" alt="IMG_5733"></a><script async src="//embedr.flickr.com/assets/client-code.js" charset="utf-8"></script>

- コントロールリングと、マルチファンクションの使い道がマジでわからない
    - そんなに使ってないからですが、どうやって活用して行こうか迷いますね。
- 動画切り替えがわかりづらい
  * MODE からの info で切り替え面倒じゃないですか、、、。これマルチファンクションとかに割り当てられる？
- 撮影後 5秒後くらいに プレビューされる
  - 撮った後にすぐにプレビューされません。SD カードも UHS-Ⅱ を使っているので遅くはないはずです。仕様？
- 動画のクロップは、動画ISオンにすると更にクロップされるので辛い気持ちになる
  - EF-S 18-135mm f3.5-5.6 を動画用に使ってみましたが、広角側で換算 35mm くらいの感じです。EF-S 10-18mm とか買ったほうがいいのか？と思うほど。
- バッテリー持ちません。室内でイベント撮影1時間くらいで電池が二個減る感じです。予備バッテリー必須ですな。
